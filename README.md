# scan-aur-package
This script will scan your aur packages and check against a blacklist for compromised packages and use local ai llama to check it further

If you want to run this script plus the AI analysis you must install llama llm version Llama 3.2 3B otherwise just skip it when it ask you.

If you want to download and install a package from the AUR just run this script and put the package name after it like this:
cd to directory where the script is and run it.
./scan-aur-package.sh fastfetch
